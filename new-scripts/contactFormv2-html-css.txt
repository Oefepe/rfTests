contactFormv2.js - HTML + CSS Requirements
================================================================================

DESCRIPTION:
Updated version of contactFormGdpr.js with stricter validation flags
(window.VALIDATION_PASSED) and generic submit button selector.

================================================================================

================================================================================
KEY DIFFERENCES FROM contactFormGdpr.js:
================================================================================

1. Part 1 listens on FORM SUBMIT, not button click
2. Requires .form-container wrapper around #contactFormContainer
3. Submit button does NOT need an ID â€” any button[type="submit"] works
4. Sets TWO validation flags: window.validationComplete AND window.VALIDATION_PASSED
5. GDPR checkboxes are NOT cleared after successful submission (unlike v1)

================================================================================
DATA ATTRIBUTES:
================================================================================

#contactFormContainer:
  - data-campaign: Campaign ID (required)
  - data-label: Label/tag ID (optional)
  - data-redirect: Redirect URL after submission (optional)

================================================================================
URL PARAMETERS:
================================================================================

Required:
  - userId
  - resourceId

Example: ?userId=123&resourceId=456

================================================================================
TWO-PART FLOW:
================================================================================

Part 1 (Validation):
  - Listens for form submit event
  - Validates all fields + GDPR consent
  - Sets window.validationComplete = true
  - Sets window.VALIDATION_PASSED = true
  - Calls Part 2

Part 2 (Contact Creation):
  - Checks BOTH validation flags before proceeding
  - Finds submit button generically (no ID required)
  - Submits to API
  - On success: redirects or clears form (GDPR checkboxes NOT cleared)

================================================================================
NOTES:
================================================================================

- This version is more flexible with button selectors
- Stricter validation gating with dual flags
- Form preserves GDPR consent state after successful submission
- Button text changes to "Submitting..." during API call
